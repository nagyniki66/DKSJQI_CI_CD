# DKSJQI_CI_CD

## CI/CD beadandó projekt
Ez a repository a CI/CD témakör gyakorlati bemutatására készült. A projekt célja egy egyszerű GitHub repository létrehozása, a Git verziókezelési folyamatok alkalmazása, valamint egy GitHub Actions workflow konfigurálása és futtatása.

## A projekt tartalma
* GitHub repository létrehozása
* Forráskód kezelése Git segítségével
* Commit és push műveletek végrehajtása
* GitHub Actions workflow létrehozása
* CI/CD folyamat demonstrálása

## Mappastruktúra
### DKSJQI_Source_code

A projekt forráskódját tartalmazó mappa.

### .github/workflows

A GitHub Actions workflow konfigurációs fájljait tartalmazó mappa.

## Workflow
A repository egy egyszerű GitHub Actions workflow-t tartalmaz, amely minden push esemény hatására automatikusan lefut. A workflow célja a CI/CD folyamat működésének bemutatása.
